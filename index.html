Cheat Sheet

HTML elements

Tag Name Description
<div> Division Creates a division.
<h1 - h6> Heading Creates a heading. Goes from h1 to h6.
<p> Paragraph Creates a paragraph.
<span> Span Creates a separate element, easier to style.
<ul> Unordered List Creates a bullet point list.
<ol> Ordered List Creates a numbered list.
<li> List Item Create an item in a list.
<br> Line Break Creates a line break to seperate elements.
<a> Link Creates a link.
<img> Image Inserts an image.
<video> Video Inserts a video.
<audio> Audio Inserts an audio.

CSS properties

Tag Name Description
<color> Color Sets the color of an element.
<background-color> Background Color Sets the background-color of an element.
<background-image> Background Image Sets the background image of an element.
<font-family> Font Family Sets the font for an element.
<font-size> Font Size Sets the size of a text.
<text-align> Text Align Sets the alignement of a text.
<margin> Margin Sets a margin around an element.
<padding> Padding Sets the padding in an element.
<border> Border Defines the border of an element.
<border-radius> Border Radius Defines the border radius of an element.
<width> Width Sets the width of an element.
<height> Height Sets the height of an element.
<display> Display Defines how an element is displayed.
